# 💰 Finance Dashboard

A modern and interactive Finance Dashboard built using Next.js and React.

## 🚀 Objective

This project was built as part of an assessment to demonstrate frontend development skills including UI design, component structure, and state management.

---

## ✨ Features

- 📊 Dashboard Overview (Balance, Income, Expense)
- 📄 Transactions List with Search
- 🔐 Role-based UI (Viewer & Admin)
- ➕ Add Transaction (Admin only)
- 📈 Data Visualization (Line Chart & Pie Chart)
- 💡 Insights Section (Top spending category)
- 💾 Local Storage Persistence
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- Next.js
- React
- Tailwind CSS
- Recharts

---

## 🧠 Key Concepts Used

- React Hooks (useState, useEffect, useMemo)
- Array Methods (map, filter, reduce)
- Conditional Rendering
- Local Storage
- Component-based Architecture
